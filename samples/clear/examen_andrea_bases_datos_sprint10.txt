SERIE I - CONCEPTOS FUNDAMENTALES

1. ¿Qué es una clave primaria?
Respuesta: Campo que identifica de manera única cada registro de una tabla.

2. Verdadero o falso: Una clave foránea puede repetir valores.
Respuesta: Verdadero

3. Complete: El proceso de organizar datos para reducir redundancia se llama __________.
Respuesta: normalización

4. ¿Qué comando SQL se usa para consultar datos?
Respuesta: SELECT

SERIE II - RESPUESTAS DIRECTAS

1. Mencione dos operaciones del álgebra relacional.
Respuesta: selección y proyección

2. ¿Qué hace un JOIN?
Respuesta: Combina filas de dos o más tablas usando una condición relacionada.

SERIE III - IDENTIFICACIÓN DE CASOS ABIERTOS

1. ¿Qué es la normalización en bases de datos?
Respuesta: Es el proceso de organizar los datos para reducir redundancia y mejorar la integridad.
