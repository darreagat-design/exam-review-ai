SERIE I - CONCEPTOS FUNDAMENTALES
Valor de la serie: 4 puntos
Cada pregunta vale 1 punto

1. ¿Qué es una clave primaria?
Respuesta correcta: Campo o conjunto de campos que identifica de forma única cada registro en una tabla.

2. Verdadero o falso: Una clave foránea puede repetir valores.
Respuesta correcta: Verdadero

3. Complete: El proceso de organizar datos para reducir redundancia se llama __________.
Respuesta correcta: normalización

4. ¿Qué comando SQL se usa para consultar datos?
Respuesta correcta: SELECT

SERIE II - RESPUESTAS DIRECTAS

2 puntos c/u

1. Mencione dos operaciones del álgebra relacional.
Respuesta correcta: selección y proyección

2. ¿Qué hace un JOIN?
Respuesta correcta: combina filas de dos o más tablas según una condición relacionada
